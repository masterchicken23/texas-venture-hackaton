# FleetCompute Frontend

React SPA for FleetCompute. Requires the backend running at `http://localhost:5000`.

## Run

```bash
npm install
npm run dev
```

Dev server runs at **http://localhost:3000** (to match backend CORS). Alternatively run with default Vite port (5173); the backend allows both origins.

## Auth (demo)

- **waymo_ops** / demo  
- **zoox_dev** / demo  
- **admin** / demo  

Auth is kept in memory only (no localStorage).
